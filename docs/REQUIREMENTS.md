# iPhone Only フェーズ

要件定義書 v4.0 の Phase 1 実装範囲。

## 実装済み

- 常設 iPhone ホスト（API / Queue / Playback / Display UI / WebSocket）
- 参加者 PWA（Home / Search / Queue / Account）
- SQLite キュー永続化
- Apple Music 検索・再生（MusicKit）
- YouTube / Spotify 検索（API キー設定時）
- サービス切替・自動次曲

## 未実装（Raspberry Pi フェーズ）

- Pi 上での制御サーバー分離
- 24時間耐久試験の自動化
- スキップ投票
